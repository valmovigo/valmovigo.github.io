Augmented Painting c

IRI https://www.perceptrum.net/ontology/augmented-painting#AugmentedPainting
Description 
An art form that combines a physical painted surface with embedded sensing technologies,
artificial intelligence, and machine learning, enabling interactive, adaptive,
and networked experiences where touch and visual interaction generate evolving behaviors.

Essential properties:
- physical painted surface
- interaction
- memory
- adaptation
- machine learning
- network communication (optional but supported)

Sound generation is NOT mandatory.
